# Victory music (optional, owner-supplied)

The celebration screen will loop a track named **`victory.mp3`** placed in this
`public/` folder. The app ships no audio of its own — drop in any file you hold the
rights to and it plays automatically when a game is won. If no file is present the
celebration screen simply stays silent. Honors the in-game mute toggle.

Do **not** commit copyrighted music to this public repository.
