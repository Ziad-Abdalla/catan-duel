Catan Duel — play online together

Free, private, no cloud. You connect over Radmin VPN (a free virtual LAN) and play in the browser. One-time: both install Radmin VPN.

① Get on the same Radmin network one time
  1. Host opens Radmin VPN → Create network → pick a name + password → share them with your friend.
  2. Friend opens Radmin VPN → Join network → enter that name + password.
  3. You should now see each other's computer with a green dot. ✅

② Host the person running the game

  1. In the repo folder, right-click RADMIN-ONLINE.ps1 → Run with PowerShell → click Yes on the prompt.
    It bridges the game to your Radmin address and opens the firewall to Radmin peers only. It prints your Radmin IP + the friend's URL.
  2. Start the two servers (in WSL): npm run party and npm run dev.
  3. Open http://localhost:5173 → Online tab → make a room → tell your friend the room code.

③ Friend joins

  1. Make sure you joined the host's Radmin network (step ①, green dot).
  2. Open a browser to http://<host's-Radmin-IP>:5173 (the host tells you this — e.g. http://26.79.157.209:5173)
  3. Click Online → type the same room code → pick the other seat. You're synced. 🎲

④ Is it safe? & closing it after

  1. Safe by design: the firewall only opens to Radmin peers (26.x) on your Private network — never the public internet. Use a password-protected network with only your friend.
  2. When you're done, the host right-clicks RADMIN-OFFLINE.ps1 → Run with PowerShell → Yes. It removes the bridge + firewall openings — nothing is exposed anymore.
  3. Optional: also right-click the Radmin VPN tray icon → Disconnect.
Trouble? 9 times out of 10 it's the friend not being on the host's Radmin network yet (no green dot). The host can ask the assistant to "check" — it can scan for the peer.
