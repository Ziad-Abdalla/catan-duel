🎲 Put Catan Duel online for everyone

No Radmin, no repo, free. Friends just open a link in any browser. ~5 minutes, once.

What you get: one permanent web address (e.g. https://catan-duel.<you>.partykit.dev) that hosts the game and the multiplayer server together. Anyone you send it to can play — phone or laptop, nothing to install. A 2-player game costs £0 (well inside the free tier).

Do this once

1
Open a terminal in the project folder, then install (first time only): 
npm install
2
Build the game: 
npm run build
Already verified working — you should see ✓ built.
3
Deploy (this is the one step only you can do — it logs into your free account): 
npm run party:deploy
A browser window opens to sign in with GitHub (free). Approve it. When it finishes it prints your permanent URL — copy it.
4
Send that URL to your friends. Both open it → Online tab → type the same room code + your names → play. Names now show on the board.

Check it works (30 seconds)

Open the URL in two browser windows (or one normal + one private/incognito). In one, go Online and note the room code. In the other, enter the same code. Make a move in one window — it should appear in the other. That confirms anyone can play.

Ship an update later

npm run build && npm run party:deploy

Same URL, updated game.

Notes

  • Local / same-house play still works with no deploy at all (PLAY.bat or npm run dev → Hotseat on one screen).
  • Why not itch.io? itch.io hosts static games only — it can’t run the multiplayer server, so online play wouldn’t work there on its own. PartyKit hosts both in one free step. (You could later host the page on itch.io and point it at a PartyKit relay, but it’s more setup for no gain.)
  • Everything else — the game, sounds, music, animations, card data — is already built and tested. Step 3’s login is the only thing that needs you.
