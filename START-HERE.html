Catan Duel

Everything is double-click. The code lives in WSL; these buttons drive it for you.

checking if the game is running…
PLAY.bat
Play on this PC

Sets up (first run) and opens the game. Use the Hotseat tab to play both sides.

PLAY-ONLINE.bat
Play online with Ziad

Opens the online control panel: it shows the firewall status, and Opens / Closes hosting in one key. Use the friend-link box below to send Ziad his link.

UPDATE.bat
Get the latest version

Pulls new changes from GitHub and re-installs. No Claude needed for updates.

STOP.bat
Stop the game

Stops the local servers. (Hosting is closed from the online control panel.)

Friend's link — enter your Radmin IP (the control panel shows it) once:

Ziad opens this after joining your Radmin network.

One-time, for online only: both of you install Radmin VPN (free) and join the same network. After that, online is just PLAY-ONLINE.bat → Open. The firewall only ever opens to your Radmin network, never the public internet, and Close makes it safe again.
